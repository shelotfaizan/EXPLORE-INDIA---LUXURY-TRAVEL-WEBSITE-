# Explore India – Luxury Travel Website

Explore India is a simple luxury-themed travel website developed as part of an academic project. The website is built using HTML and basic CSS to demonstrate understanding of web structure, layout, and styling fundamentals.

The goal of this project was to create a clean and organized travel website that presents information clearly while maintaining a premium visual theme. The design follows a luxury concept using a dark background, gold accent colors, and structured content sections.

The website includes multiple sections such as:

- A homepage introducing the travel brand
- A destinations section highlighting popular places
- A travel packages table with pricing and duration
- A booking form for users to enter trip details
- A contact section with basic company information
- Internal navigation links for smooth section access

This project focuses on core HTML concepts including headings, paragraphs, lists, tables, forms, images, and internal linking using IDs. Basic CSS has been used to improve visual appearance through background styling, text color adjustments, alignment, spacing, and layout improvements.

No external libraries, frameworks, or JavaScript have been used in this project. The purpose was to strengthen foundational knowledge of front-end web development and understand how structure and presentation work together in a webpage.

Through this project, I practiced:

- Writing clean and organized HTML code  
- Structuring content logically  
- Designing a simple but professional layout  
- Applying basic styling principles  
- Creating a user-friendly page structure  

This website can be opened directly in any browser by downloading the files and opening the `index.html` file.

This project represents a foundational step in learning web development and demonstrates practical implementation of basic HTML and CSS concepts.
